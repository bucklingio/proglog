## Grootste

```python
getal1 = float(input("Getal 1: "))
getal2 = float(input("Getal 2: "))
if getal1 == getal2:
    print("Gelijk")
else:
    if getal1 > getal2:
        print(getal1)
    else:
        print(getal2)
```
