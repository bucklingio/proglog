## Machtsverheffing 2

```python
def machtsverheffing(getal, totDeMacht):
    return getal ** totDeMacht

getal1 = int(input("Getal 1: "))
getal2 = int(input("Getal 2: "))

print(machtsverheffing(getal1, getal2))
```
